# Marketing

## Purpose

The public entry point into Ki.

Its only responsibility is to help someone understand Ki and join a meetup.

## Responsibilities

- Present the Ki brand
- Explain how Ki works
- Navigate users to Join

## Does NOT

- Handle authentication
- Handle meetings
- Handle participants
- Handle realtime
- Handle conversation logic

## Notes

Keep the page intentionally minimal.

If a section does not help someone join a meetup, it probably doesn't belong here.
