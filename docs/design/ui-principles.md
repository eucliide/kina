# Ki UI Principles

## Philosophy

Technology should quietly support real conversations.

## Design Values

- Calm
- Spacious
- Human
- Timeless
- Accessible

## Color Palette

Background: #020817

Surface: #0F172A

Accent: #2563EB

Primary Text: #F8FAFC

Secondary Text: #94A3B8

## Typography

Logo: 18px Medium

Hero: 56–64px Semibold

Body: 18px

Supporting: 16px

## Motion

Purposeful and subtle.

Maximum duration: 200ms.

## Interaction

Every screen should present one obvious primary action.
